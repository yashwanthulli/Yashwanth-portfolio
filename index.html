CNC Drawing Machine using Arduino

Abstract ‚Äî A cost-effective 2D CNC drawing machine implemented using Arduino Uno, salvaged DVD-drive stepper motors, GRBL firmware, and open-source tools (Inkscape, Universal G-code Sender). The goal is to provide an educational, modular, and low-cost plotter suitable for prototyping and learning.

  • Bluetooth module (HC-05) / optional serial interface

  • Battery pack and wiring / power management

The CNC Drawing Machine interprets G-code to create vector sketches on a 2D plane. The design emphasizes modularity and reuse of e-waste parts to keep costs low.
