CNC Drawing Machine Using Arduino

This project aims to build a low-cost CNC drawing/plotting machine using Arduino UNO, DVD stepper motors, GRBL firmware, and a servo motor for pen control. It interprets G-code generated from Inkscape and produces accurate 2D vector drawings.

Components Used:
• Arduino UNO
• DVD Stepper Motors (X & Y Axis)
• Servo Motor (Z-axis – Pen Up/Down)
• L293D or A4988 Motor Driver
• GRBL Firmware
• Recycled DVD Drive Mechanism
• Inkscape + Universal G-code Sender (UGS)
